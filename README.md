# BAKU Olympiad 2016

This `Node Js` app will scrap the players from the Official website for the Bake Chess Olympiad.

To run this code:

##### Clone the repository
```
$ git clone https://github.com/Matt-Webb/baku-olympiad
```
##### Install the dependancies
```
$ npm install
```
##### Run the app
```
$ npm start
```
